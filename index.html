Metodología & Documentación de Modelos

Fundamento académico e investigativo para el sistema predictivo de Sabana Occidente (Funza, Mosquera y Madrid).

Modelado Supervisado: Random Forest Regressor

El modelado supervisado predice de forma cuantitativa la cantidad mensual de incidentes para cada tipo de delito. Utiliza el algoritmo Random Forest Regressor que genera conjuntos de árboles de decisión para mitigar el sobreajuste y modelar interacciones no lineales entre el tiempo, las ubicaciones espaciales y los perfiles de los ciudadanos.

  • Variables de Entrada (Features): Trimestre, Día de la Semana, Año, Municipio, Género y Tipo de Delito.
  • Métrica de Desempeño: Coeficiente de determinación R² de aproximación de 96.85% en el set de pruebas (Random Forest).
  • Ajuste Estacional: Codificación mediante variables de tiempo periódicas para mitigar desfases anuales.

Agrupamiento No Supervisado: Clustering Jerárquico Aglomerativo

El agrupamiento agrupa de forma natural los meses basándose en el nivel delictivo. Utilizando Clustering Jerárquico Aglomerativo, el algoritmo identifica tres perfiles clave de vulnerabilidad territorial:

Riesgo Bajo (Clúster 0) Meses con baja presencia delictiva histórica.
Riesgo Medio (Clúster 1) Estructura delictiva moderada o focalizada.
Riesgo Alto (Clúster 2) Meses críticos con alta exposición.
